About This Project

This map displays parking citations issued in Ann Arbor, Michigan. The data is collected through automated scraping of the city's public parking citation portal.

How It Works

The citation numbers in Ann Arbor's system increment sequentially by 1. This means I can scrape the data at any moment by:

  • Starting from a known citation number and incrementing to find new citations
  • The system doesn't require authentication or special access—the citations are publicly accessible
  • Each citation contains location data, violation details, amount due, and other relevant information
  • The scraper runs automatically every couple minutes to capture new citations as they're issued

The scraped data includes location information which is then geocoded to display citations on this interactive map. Citation images are stored and compressed for efficient access. OCR (Optical Character Recognition) is used to extract text from citation images when needed, allowing for automated processing of the data contained in the receipts.

This project is open-source and available for educational purposes. The methodology demonstrates how sequential ID systems can make public data more accessible.
